# Showcase 示例图片目录

此目录用于存放 Showcase 页面的示例图片。

## 图片命名规则

请按照以下命名规则放置图片文件：

- **案例 1**（角色一致性编辑）：
  - `showcase-1-image1.jpg` - 红发女性肖像
  - `showcase-1-image2.jpg` - 中世纪盔甲女性

- **案例 2**（场景转换）：
  - `showcase-2-image1.jpg` - 咖啡店场景
  - `showcase-2-image2.jpg` - 花园场景

- **案例 3**（季节转换）：
  - `showcase-3-image1.jpg` - 雪地海滩场景
  - `showcase-3-image2.jpg` - 夏日海滩场景

- **案例 4**（风格转换）：
  - `showcase-4-image1.jpg` - 日落女性肖像
  - `showcase-4-image2.jpg` - 油画风格日落

- **案例 5**（时间转换）：
  - `showcase-5-image1.jpg` - 城市日景街道
  - `showcase-5-image2.jpg` - 城市夜景街道，有"BLUE ALICE"霓虹灯

- **案例 6**（环境转换）：
  - `showcase-6-image1.jpg` - 普通客厅阅读场景
  - `showcase-6-image2.jpg` - 魔法森林阅读场景

## 图片要求

- **格式**：建议使用 JPG 或 PNG 格式
- **尺寸**：建议 800x600 像素或更高分辨率
- **大小**：建议每张图片不超过 2MB，以优化加载速度

## 使用说明

将图片文件放入此目录后，Showcase 页面会自动加载这些图片。如果图片不存在，页面会显示占位符。
